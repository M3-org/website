About the Show

The Weekly Briefing

Cronjob is a fully-automated 3D news show covering the ElizaOS ecosystem. Every week, AI hosts Eliza and Jin break down the latest GitHub PRs, community discussions, security advisories, and market movements.

Built on PlayCanvas and rendered in real-time, Season 2 brings expanded coverage, a refreshed cast, and a cinematic weekly format. Sparty delivers the market report while Peepo keeps everyone honest with skeptical commentary.

How It's Made


Data Collection

GitHub PRs, Discord discussions, security advisories, and market data from the ElizaOS ecosystem are gathered automatically.

Script Generation

AI generates a structured session log with dialogue, scene directions, and word-level timecodes for each character.

Voice Synthesis

Each character's dialogue is converted to speech using distinct AI voice profiles.

3D Production

PlayCanvas renders the episode in real-time with dynamic cameras and cinematic lighting.
