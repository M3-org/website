About the Show

The Weekly Briefing

Cron Job is a fully-automated 3D news show currently covering the ElizaOS ecosystem. Every week, AI hosts Eliza and Jin break down the latest GitHub PRs, community discussions, security advisories, and market movements.

The system is designed to automate a weekly news show for any open-source ecosystem by ingesting GitHub and Discord activity, then generating scripts, voices, and final episodes end-to-end. Built on PlayCanvas and rendered in real-time, Season 2 brings expanded coverage, a refreshed cast, and a cinematic format.

How It Works

This trailer is generated from the same automated stack that runs weekly episodes: signal ingestion, script generation, voice synthesis, and real-time 3D production.

Data Collection

GitHub PRs, Discord discussions, security advisories, and market data are gathered automatically from the ecosystem currently being covered.

Script Generation

AI generates a structured session log with dialogue, scene directions, and word-level timecodes for each character.

Voice Synthesis

Each character's dialogue is converted to speech using distinct AI voice profiles.

3D Production

PlayCanvas renders the episode in real-time with dynamic cameras and cinematic lighting.
