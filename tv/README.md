# M3TV

DAO-native television for the decentralized age. AI-generated shows, community curated.

## Shows

- **[Cron Job](https://elizaos.news)** - Daily automated news from the ElizaOS ecosystem
- **[Clank Tank](shows/clank-tank.html)** - AI judges evaluate blockchain pitches
- **[JedAI Council](shows/jedai-council.html)** - Philosophical debates on tech and society
- **[Stonk Wars](https://twitch.tv/stonkwars)** - 24/7 live market reactions

## Structure

```
├── index.html          # Homepage
├── about.html          # About M3TV + AI pipeline
├── archive.html        # Episode archive
├── shows/              # Individual show pages
├── shows.json          # Show metadata
├── scripts/            # Dev utilities (imgen, vision, catalog)
└── resources/          # Symlinks to asset repos (gitignored)
```
